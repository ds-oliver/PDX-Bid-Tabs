# Data Dictionary: Production Dashboard Schema

Current production dashboard tables generated by `scripts/build_star_schema.py`.

## projects.csv
| Column | Type | Nullable | Definition | Source Rule |
|---|---|---|---|---|
| project_id | int | False | Project surrogate key. | Sequential key in build_star_schema.py |
| ean | str | True | Project EAN parsed from header. | compiled project_ean |
| project_name | str | True | Project title. | compiled project_name_raw |
| advertise_date | str | True | Date carried from parsed letting_date (YYYY-MM-DD). | compiled letting_date -> date cast |
| location | str | True | Normalized location code. | location_name_raw + config/location_dictionary.csv |
| solicitation_no | str | True | Solicitation identifier. | compiled solicitation_no |
| pdf_link | str | False | Source workbook filename. | compiled source_file |

## items.csv
| Column | Type | Nullable | Definition | Source Rule |
|---|---|---|---|---|
| item_id | int | False | Item surrogate key. | Sequential key in build_star_schema.py |
| project_id | int | False | FK to projects. | Project join key from compiled extract |
| line_no | str | True | Line number preserving leading zeros. | compiled line_no |
| item_description_raw | str | False | Raw item description from sheet. | compiled item_description_raw |
| specification | str | True | Parsed primary specification code. | parse_item_fields.spec_code_primary |
| alternate_specification | str | True | Parsed alternate specification code(s). | parse_item_fields.spec_code_alternates |
| pay_item_description | str | True | Parsed base item description. | parse_item_fields.item |
| supplemental_description | str | True | Parsed qualifier text. | parse_item_fields.supplemental_description |
| item | str | True | Business item key `<specification> - <pay_item_description>`. | parse_item_fields.item_display |
| quantity | float | True | Estimated quantity. | compiled quantity |
| unit_norm | str | True | Normalized unit code. | compiled unit_code_norm (fallback unit_code_raw) |

## bids.csv
| Column | Type | Nullable | Definition | Source Rule |
|---|---|---|---|---|
| bid_id | int | False | Bid surrogate key. | Sequential key in build_star_schema.py |
| item_id | int | False | FK to items. | Item join key from parsed line identity |
| contractor_name | str | False | Canonical bidder/contractor name. | compiled bidder_name_canonical |
| unit_price | float | True | Unit bid price. | compiled unit_price |
| total_price | float | True | Line total bid price (or unit_price x quantity fallback). | compiled total_price with fallback calc |
| is_winner | bool | False | True when contractor has minimum project total for that project. | derived in build_star_schema.py |
| rank | int | True | Dense rank of unit_price within item_id, excluding engineer estimate. | derived in build_star_schema.py |

## Relationships
- `projects[project_id]` 1:* `items[project_id]`
- `items[item_id]` 1:* `bids[item_id]`

## Example Join
```sql
SELECT p.project_name, i.item, b.contractor_name, b.unit_price, b.total_price
FROM bids b
JOIN items i ON i.item_id = b.item_id
JOIN projects p ON p.project_id = i.project_id;
```
